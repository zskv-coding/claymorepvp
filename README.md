# ClaymorePvP

A simple duel plugin for Spigot 1.21.

## Features
- Player dueling system
- Customizable arena spawns
- Simple configuration

## Commands
- `/duel <player>` - Challenge someone to a fight.
- `/claymorepvp setspawn1` - Set the first spawn point.
- `/claymorepvp setspawn2` - Set the second spawn point.

## Permissions
- `claymorepvp.admin` - Required for setting arena spawns.

## Setup
1. Drop the jar into your `plugins` folder.
2. Restart your server.
3. Use `/claymorepvp` to set up your arena locations.
